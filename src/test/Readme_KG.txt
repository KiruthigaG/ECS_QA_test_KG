Please make sure you have the following pre requisite to run the test:

1. Please make sure to keep a chromedriver in a location and mention the same under the Chromedriver
2. Please make sure to install the following

	-NUnit
	-NUnit3TestAdapter
	-Selenium.WebDriver
	-Selenium.WebDriver.ChromeDriver
	-Microsoft.NET.Test.Sdk 

These are the only Nuget packages i installed inorder to enable selenium.
